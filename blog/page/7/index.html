理智与情绪

Jul 19th, 2010

这篇日志可以理解为是一篇日记。

是否可以这么理解：长远的最优化的考虑都是理智的，而重视一时的感觉或者偏见是非理智的。

其实我这也只是根据自己的经验得到的结论，而这种方法本身是否理智也是个问题。

我发现我是没有办法完全理智的，尽管我希望我能够更加理智一点。我经常处于情绪化的状态，这让我十分的不舒服。

我想，人，是需要用结合理智与人本身的感情来做出决定的。至于为什么有这种需要，我想是为了寻求最佳的结果吧。

为何要寻求最佳的结果？这应该是本能，老天赋予人的礼物，抑或是原罪。

理智依靠什么？依靠科学吗？科学依靠什么？为什么是对的？因为从长远的角度能得到最好的结局吗？

用科学的方法来解释科学为什么是可行的，是可笑的。为什么是可笑的？从科学上来讲是可笑的。

Archlinux的intel网卡连接不上

Jul 17th, 2010

最新的archlinux用linux-firmware代替了以前的intel网卡驱动。

我的网卡是intel wifi 3945，装好之后怎么都不能上无限网。
提示 

SIOCSIFFLAGS: Operation not possible due to RF-kill
。

在论坛上无意中淘到了一个办法，安装rfkill。 

sudo pacman -S rfkill

然后通过以下命令查看。 

rfkill list

然后试一下以下命令，也就是解封的意思，后面的数字可能是 0 1 2 3 

sudo rfkill unblock 0

如果没问题，应该就OK了。
用

sudo ifconfig wlan0 up
测试一下。

每天只做三件事

Jun 9th, 2010

今天twitter上有一位同学发了一个tweet，分享了一篇文章，和我前几天看的一本书的观点不谋而合，很有意思的观点。

这篇文章是来自大名鼎鼎的Zen Habits博客，翻译放在译言上。

文章中提到，每天只做三件事，因为我们都很懒。作为普通人的俺们来说，就不要装逼了，大家都挺懒的，哈哈。

每天只做三件事的好处是：

  1. 只让自己做三件事，可以免去做那些繁杂的浪费时间的小事情，从而把注意力放在重要的事情上。
  2. 节约了自己的时间，让自己有时间随心所欲一下。
而作为围观群众的我们所怀疑的就是，如果重要的事情超过了三件怎么办呢？如果事情的时间不够我们完成三件怎么办呢？

在我看来，至少是以我的经验看来，这两个问题其实不是什么问题。

如果重要的事情超过了三件，那么，肯定有不重要的事情混杂在里面。比如尿急啊，给人打电话啊，去银行取钱用啊，这些都不是重要的事情，而应该是很急的事情。重要的事情有一个特征：那就是在今后很长一段时间内，这些事情做没做都会影响到我们。比如，背单词（我痛恨啊），锻炼身体（我很忙啊），与身边每个人处好关系（烦死了）等等。总之，一天能做的重要的事情，决定不会超过三件，至少以我的经验来看是如此的。如果实在是太多，比如想锻炼身体又想背单词还想泡妞还想认真学习，嗯，分几天做不会死人的。

如果时间不够我们完成三件的话，那就是划分事情有问题。做多少事情是很灵活的。比如我今天为考试复习，我不能一天复习所有的，可以选择复习一个章节。就好像唐僧取经，一天是取不完的。

还有一个问题就是，如果时间不够我们完成三件事情，那很可能是我们拖延了，做着做着就做别的事情去了。这可能是安排的事情不够重要，无法引起我们的注意力，也可能是我们抵制诱惑的能力太低了。
