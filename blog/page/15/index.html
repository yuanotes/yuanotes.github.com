VC++ 6 程序转 VS2008 的错误

Jul 4th, 2009

前一阵子在学校的博士生那里得来一个程序，是关于一个神经网络论文的实验的程序。由于那篇论文发的比较早，而现在我们现在的论文是基于那篇论文的，我们必须要重复他们的一些实验来保证那篇论文的数据的正确性。

程序是用的vc++ 6 的MFC 写的，其实最大的功能就是通过神经网络的一些算法对一些数据进行分析然后分类，也就是模式分类了。这些功能用MFC有点大材小用了，用命令行就可以了。当然这个程序也很有意思，通过MFC的消息映射来读取数据然后输出数据，但是消息映射用的是菜单上的按钮……

程序太老了，要用vs2008来跑，所以就要进行转换了。转换的问题不是很大，就是不明白MS哪来那么多的规矩，一套一套的，偏偏没事就更新换代。有关MFC的代码不是问题，但是有一点其它的问题。

首先是 include 问题，vs2008 对于 iostream 和 fstream 是不需要.h的后缀的，而且这些都在标准库中，因此需要用到std的命名空间，必须加上 using namespace std;（看情况而定了，有时候只加 std:: 更省事）

然后是 for 语句问题。现在的标准是 for 语句里面申明的变量在循环完之后就没有了，但是在vc6++里面是仍然存在的。

最后的问题就是比较让人纠结的了，执行出现类似如下错误：

这个问题在MSDN上有人提出，但是没人给出答案。我查到了CSDN上，发现也有人出现类似错误，贴下他的出错的代码。

( argc > 2
? ofstream(argv[2],ios::out|ios::binary)
:cout
) <<
( argc > 1
? ifstream(argv[1],ios::in|ios::binary)
:cin
).rdbuf() ;

return 0;
}

CSDN上没人给出好的答案，后来他自己结贴了并给出了修改后的代码。

(*( argc > 1
? &ifstream(argv[1],ios::in|ios::binary)
:&cin
)).rdbuf() ;
return 0;
} 从这里面什么也看不出来，但是总还是有一点端倪的。接下来在另一个论坛找到了相关问题的解决办法。有人回帖说这个问题可能是将一个fstream对象赋值给了另一个对象。里面也有人提到，是不是函数的形参不是通过引用传递的，这样也能出现这样的错误。

因此，我将所有的函数的形参都改过来了，最后发现，大功告成，除了一些警告之外，没有错误了。

最近的日子

Jun 19th, 2009

刚刚换了域名，也换了网站的空间，速度变得很快，一些以前的空间的问题都解决了，感觉很不错，写篇日记唠叨一下。

现在刚好放暑假了，很早以前是打算找个公司实习或者出去参加竞赛一类的活动的。现在看来，这些计划都被我放弃了。找公司实习，觉得凭自己现在的水平还没公司要我，要我的公司给的待遇我也看不上；而对于参加竞赛，我是没什么兴趣了，除了很权威的比赛之外，其它的都没什么含金量，有那个实力还不如干点别的。

暑假我就准备呆在学校，呆一个月。

第一件事就是继续帮学校的博士生做实验，帮他出论文，也学一点写论文和做实验的经验。这个实在不知道该说什么，说难，确实也不容易，那些论文资料都是我以前没有接触过的，人家研究生研究的玩意儿，我一个本科生轻易就看懂了，那我还读个屁的本科，直接搞研究去了。

还有就是继续帮学校维护网站，其实网站搭建都已经完成了，主要是输入内容比较麻烦，那些学生处的老师都只会用word，在网站上输入内容让他们觉得很繁琐。不过，那个网站我怎么看怎么觉得丑，不知道为什么，就是觉得丑死了，特别是字体按照他们的要求弄大之后，就觉得更丑了。如果像学校的官网一样设计网页，还是比较耐看的。顺便说一下，学校的官网居然是用python写的。

给学校设计网站是很无聊的，不过，学校主动提出给我们薪水，当作勤工俭学来算。这样一来就比较划算了，今天卡上无缘无故多了一千多块钱，让我觉得赚钱的感觉还挺不错。虽然这么多钱进账了，但是我也没有措手不及，反而感觉钱不够用。犹太人有句话：赚钱不难，花钱不易。

这个暑假就只有两件主要的事情了，继续维护网站和做论文实验。本来还打算利用Google App Engine自己架构一个博客程序的，现在看来，要等到我完工之后了。

嗯，又要继续记单词了，前一段时间由于考试，给自己找了个理由就中断背单词。现在则又要开始了。虽然现在对于考美国名校的研究生的热情没有以前那么强烈了，但是，总要有那么一点梦想吧。周星星说过，人要是没有理想，那跟咸鱼有什么区别？

奇怪的心理现象

Jun 2nd, 2009

现在的人，都会有一种奇怪的心理。譬如，如果听到某个离我们很遥远的人有成就了，升官了或者发财了，我们都会觉得，人家可能有那个本事或者与自己不相干。但如果这个人就在我们身边，我们往往会觉得，这个人徒有虚名，甚至还会愤愤不平找人理论。

我也会有这种心理。比如英语口语，我总是觉得，我身边的中国人英语口语不行，就是觉得没有老外那种味儿。虽然有时候自恋点觉得自己口语还行，但是也没认为自己口语有英语为母语的人的那种味道。

然而，在电视上听到中国人讲英语的时候，就感觉不同了，觉得那个说的有味道。当然，发音明显不标准的不会有这个认同感。而且发音不标准的也不容易上电视。比如现在美国的商务部长骆家辉，感觉他说出来的英语，比本地人还地道。当然他是从小在美国长大的了。

今天，我就切实感觉到了这个心理。我出教室，去上厕所，突然听到厕所旁边一个拐角有两个老外叽里咕噜在说英语，我就觉得奇怪了，这老外一般都是在她们办公室的那一楼逗留，今天跑到这里叽里咕噜讲什么。我没见到她们，但是我听的出来，是那种非英语为母语的国家的口音，虽然不是特标准，但也有点味道。

当我走过去时，就惊奇了，两个女孩都是我们学校的学生。我估计她们马上要考英语口语了，两个女孩正叽叽喳喳的练着。我当时第一感觉就是，这个女孩口语还挺标准的么。她语速很快，没有停顿，好像是背下来了。我们学校英语口语考试比较奇怪，会给学生十几二十道题目，然后让学生准备，考试的时候随机抽取一道题作答，当然接下来有自由讨论，让学生们自己讨论一个话题，老师在旁边根据每个人的表现打分。

这个心理现象还挺奇怪的，如果我身边的同学，唱个歌，或者什么表演，我都不会觉得没什么了不起，毕竟自己跟他们一起呆久了，彼此太了解了。可能我们普遍都觉得某些很成就的人都是很优秀的吧，以至于我们身边的，我们了解的，不那么优秀的人有一点突出的时候我们都觉得没什么好夸奖的。这种心理现象可能还跟成功学有那么点关系，值得研究了。
