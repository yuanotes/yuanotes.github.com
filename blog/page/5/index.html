Evince显示中文pdf乱码

Oct 3rd, 2010

在gnome里面，evince算是比较小巧的pdf阅读器了，相比于adobe reader以及xpdf这些臃肿又不美观的东西。

但是evince显示某些pdf的时候，中文是乱码或者干脆什么都没有，有个很简单的办法解决这个问题，安装poppler-data就可以了。

Ubuntu直接运行： 

sudo apt-get install poppler-data

如果还不行，那么试试这个： 

sudo apt-get install cmap-adobe-cns1 cmap-adobe-gb1 gs-cjk-resource

标题党形成的原因

Sep 27th, 2010

我们看新闻的时候常常会发现”文不对题”，就是文章的内容和标题不是一个意思，而且往往是标题比较夸大。这一般就是我们俗称的”标题党”。

然而，我还发现一个问题，那就是本该有好的标题的好文章都没有一个醒目的标题来吸引读者。google reader 上的文章就经常出现这种情况。很多很好的博文都被我忽略了，仅仅就是因为其标题毫无新意或者不够吸引眼球。

从这点来看，我觉得主要问题可能是：那些写博客的作者们并不像写新闻稿的那些记者们在乎是否有读者读他们的文章。很多习惯拟夸张标题的作者都习惯如此吸引眼球或者他们在作文课上受过专业的指导。

一般人，一般的中国人大多都不会拟标题的。因为我们并没有受过专业的指导或者培训，而中小学时期的作文的标题，都是老师拟好的。或者，让学生自由命题但是题目占分比重不大。我深以为中小学的作文课不仅要教会学生写文章，还要教会他们写醒目的标题。

然而，基于拟标题水平有差别，而读者在乎的内容又隐藏在标题之下，我觉得，一般的内容平台，最好不要只显示标题给读者让他们筛选，除非作者自信读者能够无聊到”烂新闻”也点击或者作者的标题够吸引人。放个文章摘要，图片什么的或者干脆不要隐藏全文也好。

其实，我们喜欢看醒目的标题跟我们本身也有关系。我们不仅只看好标题的文章，我们还只喜欢外观气质出众的人。我们只注重第一印象，只注重事务的外表，我们喜欢刺激，我们不喜欢枯燥，我们不喜欢观察，不喜欢细节，而且没有耐心，不敢尝试。

这些都是我们的缺点，这个时代赋予了我们这些缺点。

可视化预览编辑fonts.conf

Sep 27th, 2010

刚刚将Lucid升级到Maverick，发现无论是中文还是英文界面，中文字体总是看不顺眼。

去文泉驿网站淘了看，发现没有新字体或者什么新版本放出，但是却发现该网站给出了一个很好的工具，Fontconfig Designer，可以很方便的定制Linux下的字体。

因为现在的发行版都是用的fontconfig来定义字体的优先级以及其它的一些样式，所以该页面生成了一个XML文件的源代码，将其拷到Home目录下的.fonts.conf文件中即可。

如果是Archlinux或者Gentoo发行版，可能需要安装Ubuntu的fontconfig样式。

总之，耍Linux嘛，总得自己鼓捣鼓捣。

真正的兴趣

Aug 24th, 2010

这阵子虽然没怎么空闲，但是我却发现自己过得很空虚，没什么充实感。

其实，我挺幸运的，我这个人没事喜欢乱想，说好听点就是喜欢思考。没事我总喜欢检讨自己，这是个很好的习惯。

不过我发现了一个很不好的习惯，我只喜欢做喜欢的事情。虽然我写过一篇文章说梦想就是做自己喜欢的事情，但是，很多时候我们自己都不知道自己喜欢什么。

以前有想过，兴趣是可以培养的，甚至是可以自己培养的。我们喜欢做那些我们能做得好的事情，特别是比一般人都做得好并且受赞扬的事情。当然这只是浅层次的兴趣，就好像有人唱歌好，但并不表示他会专门去苦练自己的嗓子，有人会说话会逗人笑他也不太可能专门去练习说相声。

其实我们很容易止步于浅层次的兴趣之上，就是认为自己这个做得比别人好，以后就会一直优秀。于是梦想一个美好的未来，止步不前。

我不知道别人是如何满足目前的优秀的。有些人可能不自知，有些人可能没有勇气去进一步挖掘自己的兴趣。

真正的兴趣，是独自一个人做，自己也会沉迷的事情，就像玩游戏或者看小说一样。那是属于自己的精神世界，任何时候我们都不会去抛弃它。

去深一层的寻找自己兴趣所在是并不仅仅依赖于原始的潜层次的兴趣，还需要我们有勇气去一遍又一遍重复枯燥乏味的事情，有勇气认识自己的偏见，有勇气用理智战胜自己。
